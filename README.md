# 🇬🇧 English Version

## 🎮 Roll20 HUD

> Custom Roll20 HUD for D&D 5e — fast, compact and immersive interface designed for real gameplay on the map.

---

## 🧠 Why this project?

Roll20 is powerful, but its default interface is not optimized for fast gameplay.
This HUD brings **all essential actions directly onto the map**, reducing clicks and improving flow during sessions.

## ✨ Features

* ⚡ Compact HUD centered on screen
* 🎯 Quick access to:

  * Attributes
  * Saving Throws
  * Skills (18 buttons)
  * Combat actions
* ❤️ HP Management

  * Current / Max / Temp
  * Hit Dice (DV)
  * Quick +/- controls
* 🎲 Roll modes

  * Normal / Advantage / Disadvantage
  * Synced with character sheet
* 🧪 Dynamic Resources
* 🛡️ Global Modifiers
* ⚔️ Auto-detected combat actions
* 🎨 Clean dark fantasy UI

---

## 🚀 Installation

1. Install Tampermonkey
2. Install script:
   👉 https://raw.githubusercontent.com/Xarann/Roll20_HUD/main/roll20_hud.user.js
3. Open Roll20

### ⚠️ Important

You must roll **initiative once** after loading the page to allow the HUD to detect and sync your character data.

Without this, some values may not initialize correctly.

---

## 🧩 How it works

* **Main bar** → stats & roll modes
* **Action buttons** → navigation
* **Dynamic panel** → contextual content

---

## ⚠️ Limitations

* Depends on Roll20 DOM (may break with updates)
* Designed for D&D 5e
* Requires Tampermonkey
